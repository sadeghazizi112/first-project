Projects That I Made

  • Hello World
    This was my first webpage that I created using block elements. These are HTML elements that when displayed start on a line of their own, in effect making a "block" on the page. There are also < p > tags. These are the elements you put at the start an end of something when you want the text to be a paragraph.
  • First Webpage
    This is were I started practicing inline elements. That means things like the < h > and < strong > tags. That's heading and bolding respectively. Can't forget about the < a > tag, that's how we link things. There is WAY more than that, but lets move on for now, shall we?
  • My Favorite Things
    This was where I practiced using CSS, which stands for Cascading Style Sheet. Its basically where we make "rules" for certain kinds of text to follow. Such as "if its h2, then its blue" or "if its a paragraph, then its in italics and size 12px."
  • Paragraphs
    This was where I practiced with using Classes. That is where you make rules (see above) for entire types of text, or "classes". For an example, if you want every bold word in the HTML page to be colored red, in the css stylesheet, you create a rule stating that. The difference between this and the above lesson is that your styling entire classifications, not specifics.
  • Walrus
    This was where I practiced how to use < float > tags. It uses the css stylesheets much like the previous lessons, but alls you to decide on what side of the page the text will align. img { float: left, center, right?; }
  • Digital vs Analog
    This was the part where I got to know what Nesting is. The indendations of bulletpoint lists are a example of nesting. They are tags within the jurisdiction of other, bigger tags.
  • Greenland-Iceland
    This was where I practiced how to alter nested tags. Choosing what is affected and what isn't. Specifically, alterations to text within one nest, will not affect text outside of it.
  • My First Week
    Here was wher I was after one week of class! Quite a set up from the first day huh? For this we got to learn what Blockquotes and Jumbotrons are.Don't worry, despite its name, Jumbotrons aren't anything scary. A Jumbotron is a large box that can help draw emphasis to certain areas of your site. They're often used at the top of the page. Meanwhile a Blockquote is simply that, a block of quotation, neatly formatted.
